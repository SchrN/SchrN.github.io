Projects

RoboArena

RoboArena is a local multiplayer game, where players compete against by controlling different classes of robots in a variety of arenas. The game is built using Python, leveraging the PyQt5 framework to create a robust and interactive user interface. Roboarena was a group effort, with team members focusing on different aspects of the game.

SASL-Compiler

A compiler for the rudimentary functional programming language SASL (St Andrews Static Language), built with Python.

Project 3

In my bachelor's thesis, I modified an existing framework to ensure compatibility with the dataset used for epilepsy research. The framework was adapted to preprocess the data efficiently by cleaning and adjusting raw data. After optimizing the preprocessing stage, I tested various machine learning models to evaluate their performance and developed additional models. A key focus was on hyperparameter optimization to maximize prediction accuracy and tailor the models to the specific use case.
