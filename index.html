Bachelor Thesis - Evaluation of machine learning methods for the detection of epileptic seizures (Preprocessing)

Modification of Zhengua's work to make the framework compatible with the, at the time of the work, Klicke hier, um zu Example.com zu gelangen newest version of the TUH Seizure Corpus 2.0.0. The purpose of the framework is to preprocess raw EEG data to make it suitable for machine learning purposes. Bachelor Thesis - Evaluation of machine learning methods for the detection of epileptic seizures

The framework of ggg was modified to support the preprocessed dataset. Additionally, three new machine learning models were implemented, and the hyperparameter optimization framework Optuna was integrated. The models were trained, tested and the achieved results documented.
