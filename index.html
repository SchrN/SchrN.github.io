Projects

Project 1: Seizure Type Classification

In diesem Projekt habe ich ein Modell zur Klassifizierung von Anfallstypen auf Basis von EEG-Daten entwickelt. Die Daten stammen vom TUH Seizure Corpus. Das Modell verwendet tiefes Lernen, um verschiedene Anfallstypen zu erkennen und zu klassifizieren. Mehr Infos finden Sie hier.

Project 2: GGN for TUH Seizure Corpus

Dieses Projekt beinhaltet die Implementierung eines Graphen-Neuronalen Netzes (GGN) zur Verarbeitung und Analyse von EEG-Daten. Ziel ist es, die Leistungsfähigkeit des GGN zur Erkennung und Vorhersage von Anfällen zu demonstrieren. Mehr Infos finden Sie hier.

Der SASL-Compiler ist ein Compiler für die funktionale Programmiersprache SASL. Das Projekt umfasst die Implementierung des Compilers sowie die Optimierung und Erweiterung der Sprache. Mehr Infos finden Sie hier.
